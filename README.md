# librla_applications
This repo contains applications utilizing the 'librla': Randomized Linear Algebra library.  Users of librla are encouraged to share their applications.  Please reach out to @agillman20 do to this.

## Climate Analysis

This folder contains an illustration of the use of the 'svd_sketch' algorithm in 'librla' to analyze climate data.  In particular, it illustrates the use of 'librla' to find singular values and the dominate singular modes.   The example is taken from 

Joel A. Tropp, Alp Yurtsever, Madeleine Udell, and Volkan Cevher, Practical sketching algorithms for
low-rank matrix approximation, SIAM Journal on Matrix Analysis and Applications 38 (2017), no. 4,
1454–1485.

## Image compression

This folder contains an illustration of the use of the 'librla' package for image compression.  The examples are taken from 

Jed A. Duersch and Ming Gu, Randomized projection for rank-revealing matrix factorizations and low-rank approximations,
SIAM Review 62 (2020), no. 3, 661–682.
